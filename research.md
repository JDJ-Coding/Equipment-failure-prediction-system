## 2. 핵심 인사이트

### 2.1 정상 상태와 이상 패턴 구분 기준

#### 정상 운전 특성
- **온도**: RHK 본소성 구간 700~800°C를 안정적으로 유지 (std < 5°C)
- **전류 안정성**: 각 히터 존의 전류 변동 ±5% 이내
- **저항 안정성**: 히터 저항값이 사실상 고정 (std ≈ 0)
- **SCR 출력**: 구간별로 10~35% 범위 내에서 주기적 미세 조정
- **동작상태**: 운전 중인 모든 모터/팬 상태값 = 1 (고정)

#### 이상 패턴 기준
- 온도 급변: 분당 변화율 > ±10°C → 경고
- 전류 급등: 정상 평균 대비 > 20% 상승 → 히터 절연불량 의심
- 전류 급감: 정상 평균 대비 > 30% 감소 → 히터 단선 의심
- 저항 증가: 초기 기준값 대비 > 10% 증가 → 히터 노화
- 동작상태 0 전환: 운전 중 모터/팬 정지 → 즉각 경보

### 2.2 고장 전조증상 패턴

#### 패턴 1: 히터 저항 점진적 증가 (가장 중요)
- **원인**: 히터 코일 산화·열화로 저항값 서서히 증가
- **관찰**: RHK H4D: 1.8Ω (정상), 일부 존 2.7~4.1Ω으로 이미 높은 값 기록
- **전조**: 동일 출력(%)에서 전류가 감소하면서 전력(kW)도 감소

#### 패턴 2: 특정 히터 존 온도 불균형
- **관찰**: RHK 입측 197°C vs 출측 57°C의 극적인 온도 차이
- **정상**: 온도 프로파일이 구간별로 규칙적 패턴 유지
- **이상**: 특정 존만 온도 급변 또는 목표값(SV) 대비 PV 편차 증가

#### 패턴 3: SCR 출력 임계치 도달
- **관찰**: 일부 히터는 이미 35~91% 출력으로 높은 값
- **의미**: SCR 출력이 지속적으로 상승 → 히터 노화로 동일 온도 유지에 더 많은 에너지 필요
- **임계**: 출력 > 90% 도달 시 히터 교체 필요 신호

#### 패턴 4: 전압 변동 증가
- **관찰**: RHK#A 1U 전압 std=3.4V, 1D std=3.9V (정상 허용 범위)
- **이상**: 특정 히터 전압 std > 10V → 전기 접속 불량 또는 SCR 불량

### 2.3 가장 중요한 KPI (Key Performance Indicators)

| 순위 | KPI | 정상 기준 | 경고 기준 | 위험 기준 | 근거 |
|------|-----|----------|---------|---------|------|
| 1 | **히터 저항 [Ω]** | 기준값 ±5% | 기준값 +5~10% | 기준값 +10% 초과 | 히터 수명 직접 지표 |
| 2 | **구간별 온도 편차** | std < 3°C | std 3~8°C | std > 8°C | 균일 소성의 핵심 |
| 3 | **SCR 출력 [%]** | < 60% | 60~85% | > 85% | 용량 여유도 지표 |
| 4 | **히터 전류 변화율** | < 2%/hr | 2~5%/hr | > 5%/hr | 단선·단락 조기감지 |
| 5 | **온도 PV-SV 편차** | < 5°C | 5~15°C | > 15°C | 온도제어 성능 지표 |

### 2.4 실무 운영 인사이트

1. **예비소성(PTK) 히터 전부 비활성**: 본 데이터 수집 기간 중 예비소성로 미가동 → 라인 점검 또는 계획정비 구간일 가능성
2. **RHK 본소성 A/B 라인 모두 가동 중**: 양 라인 히터 H2~H24존 모두 정상 운전
3. **저항값 불균일성**: H5~H8 구간 저항 2.7~4.1Ω vs H2~H4 구간 1.2~1.8Ω → 킬른 후반부 히터가 더 노화된 상태
4. **입측 배기구 온도(197°C)**: 본소성로 입구에서 고온의 배기가스가 빠져나가는 것은 정상, 단 급상승 시 실(seal) 불량 신호

### 4.3 통계적 기준 (Statistical)

```python
# 롤링 Z-스코어 기반 이상 탐지

for each heater zone:
    rolling_mean = current.rolling(window).mean()
    rolling_std = current.rolling(window).std()
    z_score = (current - rolling_mean) / rolling_std

        alert("전류 상방 이상", severity="WARNING")
    if z_score < -3:
        alert("전류 하방 이상", severity="CRITICAL")  # 단선 위험

# 저항 추세 분석 (선형 회귀)
if resistance_slope > 0 and p_value < 0.05:
```

## 5. 데이터 전처리 파이프라인 설계

### 5.1 읽기 및 파싱
```python
df = pd.read_csv(filepath,
                 sep=';',
                 encoding='cp949',
                 header=[0, 1],  # 2행 헤더
                 parse_dates=['time'],
                 date_format='%d.%m.%Y %H:%M:%S.%f')
```

### 5.2 피처 엔지니어링
```python
# 1. 롤링 통계
df['H_current_rolling_mean'] = df['current'].rolling(60).mean()
df['H_current_rolling_std'] = df['current'].rolling(60).std()

# 2. 변화율
df['resistance_slope'] = df['resistance'].diff(60)  # 1시간 변화량

# 3. Z-score
df['current_zscore'] = (df['current'] - rolling_mean) / rolling_std

# 4. PV-SV 편차
df['temp_deviation'] = df['temp_pv'] - df['temp_sv']

# 5. 출력 대비 저항 이상도
df['resistance_anomaly'] = df['resistance'] / df['resistance'].rolling(1440).min() - 1
```

### 5.3 이상치 처리
- **65535 (PLC 통신 오류값)**: → NaN으로 대체 후 선형 보간
- **음수 온도**: → NaN 처리
- **전류 스파이크** (평균 ±5σ 초과): → 이전값으로 대체 (forward fill)
