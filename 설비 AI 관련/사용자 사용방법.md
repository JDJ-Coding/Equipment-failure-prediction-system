# 소성로 설비 이상예지 대시보드 — 사용자 가이드

> 대상: 설비관리 현장 담당자 (비개발자)

---

## 대시보드란?

소성로에 설치된 수백 개의 센서 데이터를 자동으로 분석해서
**"어떤 설비가 얼마나 위험한지"**, **"어떤 센서에 문제가 있는지"**를
한눈에 보여주는 화면입니다.

---

## 1. 대시보드 접속

담당 IT 관리자가 알려준 주소를 인터넷 브라우저(Chrome, Edge 등)에 입력합니다.

```
예시: http://localhost:8501
     또는 http://192.168.1.100:8501  (사내 네트워크 주소)
```

---

## 2. 데이터 파일 올리기

화면 왼쪽 **사이드바**에서 파일을 업로드합니다.

### 방법

1. 왼쪽 사이드바의 **"📂 데이터 파일 선택"** 버튼 클릭
2. 분석할 CSV 파일 또는 Excel 파일을 선택
3. 잠시 기다리면 자동으로 분석이 시작됩니다

### 지원 파일 형식

| 형식 | 예시 |
|------|------|
| CSV/TXT | `소성로 히터 csv파일.txt` |
| Excel | `본소성 히터 csv 파일.xlsx` |

> **파일을 올리지 않아도** 기본 데이터로 자동 실행됩니다.

---

## 3. 설비 선택하기

왼쪽 사이드바의 **"🏭 분석할 설비"** 드롭다운을 클릭하면
데이터가 있는 설비 목록이 나타납니다.

```
예시: RHK-A, RHK-B, PTK-히터, NCF1 ...
```

- 원하는 설비를 선택하면 오른쪽 화면이 자동으로 업데이트됩니다
- **데이터가 없는 설비(미가동)는 목록에 표시되지 않습니다**

---

## 4. 건강도 게이지 보는 법

화면 상단 왼쪽에 **원형 게이지**가 표시됩니다.

| 색상 | 점수 | 의미 | 조치 |
|------|------|------|------|
| 🟢 초록 | 80~100점 | **양호** — 정상 운전 중 | 정기 점검 유지 |
| 🟡 노랑 | 60~80점 | **주의** — 일부 센서 이상 | 2주 내 점검 |
| 🟠 주황 | 40~60점 | **경고** — 점검 필요 | 1주 내 점검 |
| 🔴 빨강 | 0~40점 | **위험** — 즉시 조치 | 즉시 정비 |

---

## 5. 위험 센서 TOP 5 확인하기

건강도 게이지 오른쪽에 **가장 위험한 센서 5개**가 표시됩니다.

```
🔴 1. SCR H1U 전류(A)  [전류(A)]
      건강도: 32.5/100  |  이상 감지: 87건

🟠 2. SCR H2M 전압(V)  [전압(V)]
      건강도: 55.1/100  |  이상 감지: 42건
```

- 건강도가 낮을수록 위험
- "이상 감지 건수"는 오늘 데이터(1,440분)에서 이상 판정된 횟수

---

## 6. 시계열 차트 보기

화면 중간의 **시간 그래프**에서 센서 값의 변화를 확인합니다.

### 센서 카테고리 선택

차트 위의 선택 박스에서 보고 싶은 카테고리를 선택합니다:
- `온도(℃)` — 히터 온도
- `전류(A)` — SCR 전류
- `전력(kW/W)` — 전력 소비량
- `출력률(%)` — 제어 출력
- 기타 (압력, 유량, 전압 등)

### 차트 조작 방법

| 동작 | 방법 |
|------|------|
| 특정 구간 확대 | 마우스로 드래그하여 범위 선택 |
| 이전 화면으로 | 더블클릭 |
| 그래프 저장 | 차트 오른쪽 위 📷 아이콘 클릭 |
| 센서 숨기기/보이기 | 범례(그래프 설명) 클릭 |

### 빨간 배경 음영의 의미

차트의 **빨간 배경 구간** = 이상 감지 시간대입니다.
해당 시간에 하나 이상의 센서에서 이상 값이 감지된 것입니다.

### 롤링 평균이란?

센서 값의 **순간 변동을 부드럽게 평균낸 값**입니다.
왼쪽 사이드바에서 **10분 / 30분 / 60분** 중 선택할 수 있습니다.
숫자가 클수록 더 부드러운 그래프가 표시됩니다.

## 7. 한국어 분석 리포트 보기

차트 아래의 **"📋 한국어 분석 리포트"** 박스를 클릭하면 펼쳐집니다.

```

[RHK-A] 건강도: 61.3/100 | 상태: 주의 – 일부 센서 이상 감지
분석 기간: 2025-09-14 00:16 ~ 23:59 | 전체 이상감지율: 8.2%
트렌드: ⚠ 악화 추세 (최근 이상률 상승)

▶ 주요 위험 센서 TOP 3
  1. SCR H1U 전류(A) [전류(A)] | 건강도 32.5/100 | 이상 87건 (6.0%)
  2. SCR H2M 전압(V) [전압(V)] | 건강도 55.1/100 | 이상 42건 (2.9%)
  3. DPT 압력 [압력] | 건강도 68.0/100 | 이상 18건 (1.3%)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 8. 전체 설비 정비 우선순위 테이블

화면 하단에 **모든 설비의 건강도와 정비 우선순위**가 표시됩니다.

- **건강도 낮은 순서**로 정렬되어 있어 제일 위에 있는 설비가 가장 급합니다
- 색상으로 상태를 바로 확인할 수 있습니다:
  - 🟢 초록 배경 = 양호
  - 🟡 노랑 배경 = 주의
  - 🟠 주황 배경 = 경고
  - 🔴 빨강 배경 = 위험

> 왼쪽 사이드바의 **"전체 설비 정비 우선순위"** 체크박스를 켜야 보입니다.

---

## 9. PPT 5줄 요약 보기

회의나 보고서에 사용할 수 있는 **핵심 5줄 요약**입니다.

1. 왼쪽 사이드바에서 **"PPT 5줄 요약"** 체크박스 체크
2. 화면 하단에 요약 텍스트가 표시됩니다:

```
[소성로 설비 이상예지 분석 요약] 분석일시: 2025-09-14 15:30
▶ 총 2개 설비 분석 완료, 평균 건강도 72.4/100
▶ 즉시 점검 필요 설비 0개 (건강도 60점 미만)
▶ 최우선 정비 대상: RHK-A (건강도 61.3, 주요 이상 센서: SCR H1U 전류(A))
▶ 최양호 설비: RHK-B (건강도 83.5)
```

## 10. 화면 저장 / 공유 방법

### 시계열 차트 이미지 저장

차트 오른쪽 상단의 **📷 카메라 아이콘** 클릭 → PNG 파일로 저장

### 화면 전체 캡처

- Windows: `Windows키 + Shift + S` (화면 캡처 도구)
- macOS: `Command + Shift + 4`

### 테이블 데이터 복사

정비 우선순위 테이블 위에서 마우스로 선택 → 복사(Ctrl+C) → Excel 붙여넣기
